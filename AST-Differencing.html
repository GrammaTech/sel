3.9 AST Differencing

The software-evolution-library/ast-diff library provides for the differencing of software objects. A general tree differencing algorithm (originally adopted from http://thume.ca/2017/06/17/tree-diffing/#a-tree-diff-optimizer) is used to difference tree genomes. Optimizations are made which assume the common cases of differences typically found in software development in which most top-level subtrees have no differences. This library is used to define the clang-diff and lisp-diff command-line executables. These executables provide for AST-level differencing and may be used as a replacement for common line- or word-based differencing tools during software development.

• clang-diff:  Command-line AST-differences for c/c++ files
• lisp-diff:  Command-line AST-differences for Lisp files
