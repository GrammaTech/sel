3.1 Test Suites

SEL incorporates representations for runnable unit tests. This allows individual components (such as Condition Synthesis and Tracing) and evolution projects to determine whether unit tests are passing, and if their behavior has changed.

3.1.1 API Support For Unit Tests

API support for unit tests is documented in section API | Test Suites. We provide a brief overview here.
