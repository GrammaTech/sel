5.1 Installation

The Software Evolution Library (SEL) requires one of the following supported lisp distributions. (Relatively recent versions of these implementations are required because of the use of new features of ASDF/UIOP.)

ImplementationMinimum VersionURL
Steel Bank Common Lisp (SBCL)1.4https://www.cliki.net/SBCL
Clozure Common Lisp (CCL)1.12-devhttps://www.cliki.net/CCL

The source code for SEL is available at https://github.com/GrammaTech/sel. The following Common Lisp packages are required by this library, all of which are installable through Quicklisp (see http://www.quicklisp.org/).

  • alexandria
  • metabang-bind
  • curry-compose-reader-macros
  • split-sequence
  • cl-ppcre

The following additional Common Lisp packages are required by this library but are not (yet) installable through Quicklisp. These should be downloaded into your Quicklisp local-projects/ folder.

  • trace-db (available from https://github.com/GrammaTech/trace-db)
  • command-line-arguments (available from https://github.com/eschulte/command-line-arguments)

Depending on the type of software object used the following may also be required.

  • clang-mutate for clang object support (available from https://github.com/GrammaTech/clang-mutate).
  • cil-mutate for cil object support (available from https://github.com/eschulte/cil-mutate).
  • llvm-mutate for llvm object support (available from https://github.com/eschulte/llvm-mutate).

Install llvm-mutate, cil-mutate and clang-mutate as per the instructions in their respective repositories. (Note, it is only necessary to install the tools for the specific software backends you intend to use.) Install SEL by cloning the repository into your quicklisp local-projects directory and then running: 

    (ql:register-local-projects)
    (ql:quickload :software-evolution-library)
    (ql:quickload :software-evolution-library/utility)
