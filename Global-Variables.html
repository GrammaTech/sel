2.4 Global Variables

The following global variables are exposed for configuration and interaction with heuristic search processes. This table contains summary information only, see the SEL API section for full details.

*population*A list of the software objects currently known to the system.
*max-population-size*Maximum allowable population size.
*generations*The running generation count.
*tournament-size*Number of individuals to participate in tournament selection.
*tournament-eviction-size*Number of individuals to participate in eviction tournaments.
*tournament-selector*Function used to select winners of a tournament.
*tournament-tie-breaker*Function used to break ties when a tournament has multiple winners.
*tie-breaker-predicate*Function to compare two tie breaker values to select which is preferred.
*fitness-predicate* Function to compare two fitness values to select which is preferred.
*cross-chance*Fraction of new individuals generated using crossover rather than mutation.
*mut-rate*Chance to mutate a new individual.
*fitness-evals*Tracks the total number of fitness evaluations performed.
*running*True when a search process is running, set to nil to stop a running search.
