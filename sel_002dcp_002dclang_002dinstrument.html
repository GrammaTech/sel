C.44 sel-cp-clang-instrument

Class: clang-instrumenter

Class precedence list: clang-instrumenter, instrumenter, standard-object, t

Slots:

  • names — initarg: :names; reader: software-evolution-library/components/clang-instrument::names; writer: (setf software-evolution-library/components/clang-instrument::names)

    Mapping of names to indices.

  • types — initarg: :types; reader: software-evolution-library/software/clang:types; writer: (setf software-evolution-library/software/clang:types)

    Mapping of trace type strings to indices.

  • type-descriptions — initarg: :type-descriptions; reader: software-evolution-library/components/clang-instrument::type-descriptions; writer: (setf software-evolution-library/components/clang-instrument::type-descriptions)

    Mapping of type descriptions to indices.

  • ast-ids — initarg: :ast-ids; reader: software-evolution-library/components/clang-instrument::ast-ids; writer: (setf software-evolution-library/components/clang-instrument::ast-ids)

    Mapping of ASTs to trace ids.

Handles instrumentation for clang software objects.

Function: clang-instrument source &key help quiet verbose load eval out-dir read-seed save-seed compiler flags save-original trace-file variables scope strings point exit

Instrument source.

Keyword arguments:

  • help (boolean) display help output
  • quiet (boolean) set verbosity level to 0
  • verbose (integer) verbosity level 0-4 (default: 2)
  • load (string) load file as lisp code (may be used to define scions)
  • eval (string) eval string as lisp code
  • out-dir (string) write final population into dir
  • read-seed (string) load random seed from file
  • save-seed (string) save random seed to file
  • compiler (string) use cc as the C compiler (default: "clang")
  • flags (string) comma-separated list of compiler flags
  • save-original (boolean) also save a copy of the original
  • trace-file (string) instrumented to write trace to file (default: "stderr")
  • variables (boolean) write unbound variables to trace
  • scope (boolean) write in-scope variables to trace
  • strings (boolean) trace string variable values, dangerous
  • point (string) instrument to print string at ast num
  • exit (boolean) instrument function exit
Generic Function: instrument-c-exprs instrumenter exprs-and-types print-strings

Generate C code to print the values of expressions. exprs-and-types is a list of (string . clang-type) pairs. Returns a list of strings containing C source code.
