2.1.4 Java with java

Implements the core functionality of the software-evolution-library for class files written in java. The java versions that are supported relies heavily on an input’s success of parsing with Java Parser.

The core functionality is supported by the command line tool java-mutator.

Note: certain methods in this class are over-loaded to provide consistency with other software objects, but are not used in this implementation.
