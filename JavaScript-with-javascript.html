2.1.5 JavaScript with javascript

Implements the core functionality of the software-evolution-library for files written in JavaScript. The core functionality is supported by the command line tool acorn, which parses the code into ASTs.

Additionally, the prettier command line JavaScript code formatter is required to support code formatting (see see Generic-Function format-genome).

As with clang software objects, javascript software objects are a parsed, AST representation. All of the generic methods for parseable software objects have been implemented, allowing for a rich variety of manipulations and tests.

Every AST in a javascript software object contains an AST class field with the AST type (e.g. BlockStatement, ForStatement). Additional fields given by the acorn parser can be found in an association list on the AST’s aux-data field. For more information on the ASTs, see esprima.

Note: certain methods in this class are over-loaded to provide consistency with other software objects, but are not used in this implementation.
