C.16 sel-sw-csurf-asm

Class: csurf-asm

Class precedence list: csurf-asm, asm-heap, asm, simple, software, oid-object, file-w-attributes, file, standard-object, t

Slots:

  • assembler — initarg: :assembler; reader: software-evolution-library/software/csurf-asm:assembler; writer: (setf software-evolution-library/software/csurf-asm:assembler)

    Assembler to use for assembling.

  • linker — initarg: :linker; reader: software-evolution-library/software/asm::linker; writer: (setf software-evolution-library/software/asm::linker)

    Linker to use for linking.

  • flags — initarg: :flags; reader: software-evolution-library/software/asm::flags; writer: (setf software-evolution-library/software/asm::flags)

    Flags to use for linking

  • asm-flags — initarg: :asm-flags; reader: software-evolution-library/software/csurf-asm:asm-flags; writer: (setf software-evolution-library/software/csurf-asm:asm-flags)

    Flags to pass to assembler.

  • redirect-file — initarg: :redirect-file; reader: software-evolution-library/software/csurf-asm:redirect-file; writer: (setf software-evolution-library/software/csurf-asm:redirect-file)

    CodeSurfer redirect file to redirect elf copy relocations.

  • linker-script — initarg: :linker-script; reader: software-evolution-library/software/csurf-asm:linker-script

    CodeSurfer linker script to pin section locations.

  • weak-symbols — initarg: :weak-symbols; reader: software-evolution-library/software/csurf-asm:weak-symbols; writer: (setf software-evolution-library/software/csurf-asm:weak-symbols)

    Symbols to weaken with ‘elf-weaken-gmon-start’.

  • linked-files — initarg: :linked-files; reader: software-evolution-library/software/csurf-asm:linked-files; writer: (setf software-evolution-library/software/csurf-asm:linked-files)

    List of additional libraries to link.

Software object for ASM generated by CodeSurfer.

Variable: *dynamic-linker-path*

Path to the dynamic linker on this system.

Variable: *elf-copy-redirect-path*

Path to elf_copy_redirect (or just the name if it’s on the path).

Variable: *elf-edit-symtab-path*

Path to elf_edit_symtab64 or 32 (or just the name if it’s on the path).
